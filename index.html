Lampa Plugins

Плагины для медиа-плеера Lampa. Установка: Настройки → Расширения → Добавить плагин → вставить URL.

🛒 My Store — магазин плагинов (рекомендуется)

Одна ссылка — полный доступ ко всем плагинам через нативный UI Lampa. Добавляет пункт «Мои плагины» в настройки, открывает встроенный магазин расширений с категориями и install/remove. Автоматически ставит сборщик логов на первом запуске.

https://vahagn-99.github.io/lampa/app/store.js

Источник сегментов

Автор: @vahagn_99 · v2.0.0
